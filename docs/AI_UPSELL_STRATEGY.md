# AI Upsell Strategy — Listen, Did You Know?

This app should use AI as a premium curiosity layer. The base product is simple: one daily fact, a source link to Justin's Short, sharing, and archive discovery. AI should unlock personalization, explanation, trivia, and archive search.

## Core Principle

```text
Base product = daily fact + source link + subscribe/share CTA
AI upsell = explain, personalize, quiz, organize, and search the fact vault
```

## Premium Consumer Features

| Feature | Description | Tier |
|---|---|---|
| Ask the Fact Vault | Ask questions across approved facts and receive source-backed answers | Pro |
| Explain This Fact | Expand a short fact into background, meaning, and related context | Plus |
| Daily Curiosity Briefing | Personalized facts by interest: animals, history, science, food, weird facts, etc. | Plus |
| Trivia Mode | Turns facts into quizzes, challenge streaks, and shareable questions | Pro |
| Themed Fact Packs | AI groups facts into collections for learning and sharing | Plus |
| Fact Memory | Tracks what the user has learned and resurfaces related facts later | Pro |

## Creator / Partner Features

| Feature | Description | Tier |
|---|---|---|
| Shorts Extraction Copilot | Processes new Shorts and extracts candidate facts | Creator Pro |
| Topic Classifier | Categorizes facts into themes for discovery and sponsorship | Creator Pro |
| CTA Optimizer | Suggests Subscribe, Merch, Watch Short, Newsletter, or Sponsor CTA | Creator Pro |
| Quality Filter | Flags weak or duplicate facts before publishing | Creator Pro |
| Sponsor Fit | Maps fact topics to sponsor categories such as education, games, science, trivia, or family | Enterprise |

## Subscription Concept

```text
Free

Plus

- Trivia mode
- Fact memory
- Advanced search

Creator/Partner
- Shorts extraction
- Topic classification
- CTA optimization
- Analytics
```

## Guardrails

- AI must not invent facts and attribute them to Justin.
- AI answers should reference approved source facts and original Shorts.
- Original facts should remain visually separate from AI-generated explanations.
- Voice, likeness, or persona features should require authorization.
- AI-generated explanations should avoid presenting uncertain information as verified fact.

## Why It Matters

The daily fact product gets people to return. AI turns the archive into a paid learning and trivia experience:

```text
Shorts archive → extracted facts → searchable vault → AI explainer/trivia engine → premium subscription
```
